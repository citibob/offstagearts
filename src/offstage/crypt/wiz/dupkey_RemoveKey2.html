Remove Duplicate Key

Done writing the duplicate key.

  • 1. Remove the key. WARNING: Do not just pull the key out! Removal must be done as follows:
    • Windows: Use the "safely remove media" tool in the system tray.
    • Macintosh: Use the eject button in a Finder window.
  • 2. Put the key in a safe place. Payment details cannot be retrieved if it is lost.
